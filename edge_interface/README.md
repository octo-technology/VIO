# vio-interface

## Prerequisites

### Linux setup
```bash
sudo apt update
sudo apt install nodejs
```

### Mac setuo
```bash
brew update
brew install node
```

## Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
